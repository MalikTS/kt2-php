# Простая PHP Капча (CAPTCHA)

Этот проект представляет собой простую и эффективную систему капчи на PHP для защиты ваших веб-форм от спам-ботов. Капча генерирует случайный код, отображает его в виде изображения и требует от пользователя ввести этот код для подтверждения того, что он человек.

## Возможности

*   **Генерация случайного кода:** Создает уникальную строку из букв и/или цифр при каждой загрузке страницы.
*   **Визуальное представление:** Отображает сгенерированный код в виде искаженного изображения (PNG), чтобы усложнить его распознавание ботами.
*   **Обновление кода:** Кнопка обновления позволяет пользователю получить новый код, если текущий сложно разобрать.
*   **Проверка на стороне сервера:** Введенный пользователем код проверяется на сервере с помощью PHP.
*   **Понятная обратная связь:** Пользователь получает четкие сообщения об успехе или ошибке.
    *   **Успех:** При правильном вводе кода появляется сообщение "Форма успешно отправлена!".
    *   **Ошибка:** При неправильном вводе или пустом поле ввода форма подсвечивается красным, и появляется сообщение "Пожалуйста введите код!".

## Требования

*   PHP 7.0 или выше
*   Библиотека GD (обычно включена по умолчанию в большинстве хостингов)
*   Веб-сервер (например, Apache, Nginx)


## Использование

1.  Пользователь видит форму с изображением капчи.
2.  Пользователь вводит символы, которые он видит на изображении, в текстовое поле.
3.  Пользователь нажимает кнопку "Отправить".
4.  Скрипт на сервере проверяет введенный код.
    *   Если код верный, выполняется основное действие формы (например, отправка данных).
    *   Если код неверный, пользователю показывается сообщение об ошибке, и ему предлагается попробовать снова.
